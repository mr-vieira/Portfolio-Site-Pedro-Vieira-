Now "what"?

With a primary Persona, User Stories and a refined Problem Statement, I was now able to think of new functionalities or features aimed at solving real problems for real people.

Hypothesis

üí° custom study plan

By creating a feature that allows setting a custom study plan, for Julian we‚Äôll achieve a consistent record of tasks and exercises completed over a month, thus reflecting a new learning habit.

üí° everyday vocabulary lessons

üí° custom vocabulary

By building a feature that allows Julian to save custom words, phrases and idioms, we will allow Julian to engage with his learning process, create custom content and remember his new learnings in real context.
